**Description:**
The goals of the project were: 1) to train to prepare data for further analysis 2) to train to explore the data by finding relations between different values.  We were given a dataset containing information on borrowers' marital status, number of children, loan purposes and we were asked to analyse if there were an impact of such factors on  borrowers' default on a loan.

**Libraries:**
*pandas*, *numpy*, *nltk*
